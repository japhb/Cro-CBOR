NAME
====

Cro::CBOR - Support CBOR in Cro for body parsing/serializing

SYNOPSIS
========

```raku
use Cro::CBOR;
```

DESCRIPTION
===========

Cro::CBOR is a set of extensions to `Cro::HTTP` and `Cro::WebSocket` to support using CBOR alongside JSON as a standard body serialization format.
